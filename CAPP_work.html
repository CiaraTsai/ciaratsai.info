Wheel-Design Computer-aided Process Planning Cloud Service (Windows Azure - Public Cloud)

輪框設計電腦輔助製程規劃雲端服務系統

台北國際工具機展、AIM-HI, 2013

作品簡介

WD-CAPP是一套協助製程工程師在大量生產輪框前，智能化推薦工具機的刀具加工輪框，並且模擬虛擬工具機的切削過程，評估與驗證是否會造成碰撞的雲端服務系統。並實際與製造生產線進行整合，工具機與輪框資料來源是與嘉義發得科技合作並取得。 Implement with WebService(Java, C#), Windows Azure (public cloud), Silverlight, Protege…etc.

主要技術工作
  • Design (UI)
  • Front-end development
  • Service Kernel development
  • Automated
技術層面
  • Windows Azure (public cloud)
  • Windows Communication Foundation (WCF)
  • Silverlight

  • Auto-Scaling
  • MySQL (Database)
  • Azure Storage - BLOB
  • Azure Storage - TABLE
開發過程
1. 與機械系團隊進行需求蒐集。
2. 與發得科技員工開會需求蒐集與確認流程
3. 與IT團隊規劃雲端服務框架與功能。
4. 規劃與設計系統。
5. 利用軟體工程概念規劃資料庫。
6. 利用Java, C#等語言將製造元件雲端服務化。
