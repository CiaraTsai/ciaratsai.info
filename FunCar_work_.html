Gofun

設計及實作整合異質性資訊設備之旅遊平台

作品簡介

現在科技進步如此神速，利用科技與網路結合將旅遊過程完整記錄，一定少 不了能夠的補捉當時回憶的影像記錄，此系統是利用3G 智慧型手機，系統平台是使用Android 平台開發出一套透過Web 2.0 技術之旅遊影像記錄器，這可以讓使用者隨時利用3G 智慧型手機或是相機記錄下不同的旅遊回憶，並透過Web 2.0 的技術來管理我們的影像記錄以及路徑。

主要技術工作
  • Design (Logo, UI, ...)
  • Front-end development
技術層面
  • Java (Android sdk 2.3)
  • HTML
  • PHP
  • Javascript
  • Ajax
  • jQuery
  • MySQL (Database)
  • SQLite (Storage)
開發過程
1. 規劃系統與需求探討。
2. 與教授們討論適當步驟。
3. 利用軟體工程概念規劃資料庫。
4. 利用Android使用Java語言實作出此系統，並透過JavaScript動態顯示地圖。
