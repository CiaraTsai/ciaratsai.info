ICYE

交青管理系統

大學二年級下學期所接的CASE, 2010 - 2011

作品簡介

ICYE協會內部想要有一個管理志工通訊錄的系統，並且有簡易查詢的功能，因此系統是用來協助ICYE協會管理志工活動紀錄與基本資料，加上有簡易的查詢功能方便了解志工所有記錄的資料。

主要技術工作
  • Design (UI, Database...)
  • Front-end development
  • Database Maintenance
技術層面
  • HTML
  • PHP
  • Javascript
  • Ajax
  • jQuery
  • MySQL (Database)
開發過程
1. 規劃系統與需求探討。
2. 規劃資料庫與系統架構。
3. 利用PHP與MySQL資料庫實作。
4. 利用Jquery將版面操作流程變的實用。
5. 使用JavaScript與Ajax技術做動態互動效果以及查詢。
