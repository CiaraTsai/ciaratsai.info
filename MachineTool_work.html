Recommend Machine Tool Cloud Service

機台推薦雲端服務系統

台中國際工具機展、AIM-HI, 2012 - 2013

作品簡介

Recommend Machine Tool Cloud Service是一套智能化推薦製程工程師在工廠內大量的工具機中該選擇可加工且合適的工具機來進行加工。我們也透過虛擬工具機的模擬評估，來進行加工前的驗證，若無加工前評估，避免造成工具機撞機而導致公司損失。 Implement with WebService(Java, C#), Windows Azure (public cloud), Silverlight, Protege…etc.

  • Design (UI)
  • Front-end development

開發過程
1. 與機械系團隊進行需求蒐集。
2. 與福裕公司員工開會需求蒐集與確認流程
3. 與IT團隊規劃雲端服務框架與功能。
4. 規劃與設計系統。
5. 利用軟體工程概念規劃資料庫。
6. 利用Java, C#等語言將製造元件雲端服務化。
