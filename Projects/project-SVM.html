Introduce

Automated Software Verification是一套協助使用雲端開發人員自動化檢測的一套系統，其是基於工作流程與我們設計的雲端開發框架進行SaaS服務的檢測。只要使用我們提出的框架進行開發，就可額外進行雲端服務軟體的檢測。我們使用WD-CAPP作為實際自動化檢測雲端服務的應用。

開發過程

  1. 進行需求蒐集。
  2. 設計開發原則。
  3. 利用軟體工程概念規劃資料庫。
  4. 將WD-CAPP雲端系統導入Workflow技術。
  5. 透過平行處理的Workflow進行同時運作WD-CAPP雲端服務與SVM檢測服務
  6. 利利用Java Web Service進行檢測雲端服務核心開發。
